How to use:

Registration and login:

  1. Fill in the fields in the registration form;
  2. Make sure the fields are unique;
  3. After successful registration, please log in.

Add notes:

  1. After successful login you will be redirected to notes list;
  2. On this page you can see all notes;
  3. In the navigation panel, you can select "Add new note" to add a new note.

Editing of notes:

You have permission: admin or uploader

admin
- have all possible abilities: for notes(add, edit, delete), for user management (edit name and email, change permission of users);
user
- see all notes and add new;

For other details, please visit README.md
